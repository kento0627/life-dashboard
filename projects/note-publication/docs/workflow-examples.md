# 対話型執筆フロー - 具体例

## 概要
このドキュメントでは、Claude Codeを使った対話型の記事執筆フローを
具体的な会話例で説明します。

---

## 例1: 物件探しの記事

### ターン1: トピック提示
```
You: 物件探しについて記事書きたいんだよね
```

```
Claude: いいね！物件探しか。いくつか聞かせて。

1. この記事で一番伝えたい「結論」は何？
   （例: ネットで探すな、とか、こういう方法がある、とか）

2. 具体的にどんな物件を探した時の話？
   （tokoro hotel Isumi？それとも別の？）

3. 読者は誰を想定してる？
   （不動産投資したい人？民泊やりたい人？）

あと、story-bankを見たら関連エピソードがあったよ:
- [STORY-001] 関東全自治体への電話営業
- [STORY-002] キャッシュなし・信用なしの状態

この辺りの話を絡める感じ？
```

### ターン2: 回答 → 深掘り
```
You: うん、自治体に電話しまくった話。結論は「ネットに載ってない物件を
どう見つけるか」かな。昔の自分みたいに、物件探しで悩んでる人向け。
```

```
Claude: なるほど、自治体電話の話ね。もう少し聞かせて。

1. 「関東全ての自治体」って、具体的に何件くらい電話した？
2. 電話した時、最初なんて言って切り出した？トークの流れある？
3. 成功率はどのくらいだった？（100件中何件話聞いてもらえた、とか）
```

### ターン3: 回答 → 構成提案
```
You: 200件くらいかな。最初は「空き家対策のご担当の方いらっしゃいますか」
って聞いて、いきなり物件くださいとは言わない。相談ベースで入る感じ。
成功率は...まともに話聞いてもらえたのは30件くらいかな。
```

```
Claude: OK、だいぶ見えてきた。こんな構成でどう？

---
**タイトル案:** 
「ネットに載ってない物件」を見つける方法。僕は関東200自治体に電話した。

**構成:**
1. **リード**: ネットで物件探しして満足してない？お宝はそこにはないよ
2. **導入**: 26歳、キャッシュなし信用なしで物件探しを始めた話
3. **本文①**: なぜ「自治体に電話」という発想に至ったか
4. **本文②**: 実際のトークスクリプト（空き家対策の担当者への切り出し方）
5. **本文③**: 200件電話して30件話を聞いてもらえた。その30件の共通点
6. **まとめ**: ネットに出てない情報を取りに行く姿勢。作業ゲーを楽しめ

**使うエピソード:**
- [STORY-001] 関東全自治体への電話営業
- [STORY-002] キャッシュなし・信用なしの状態
- [新情報] 200件→30件の成功率、トークスクリプト

---
この方向でいい？修正あれば言って。OKなら下書き書くよ。
```

### ターン4: 承認 → 生成
```
You: いいね、書いて！
```

```
Claude: 了解、書くね。

[下書き生成中...]

---

できた！ `articles/property-hunting-200calls/draft.md` に保存したよ。

あと、今回の対話で新しいエピソードが出てきたから、
story-bankに追加しておくね:

---
### [STORY-015] 自治体電話のトークスクリプト
- **時期**: 2024年（26歳）
- **内容**: 「空き家対策のご担当の方いらっしゃいますか」から始める
- **ポイント**: いきなり物件くださいと言わない、相談ベースで入る
- **成功率**: 200件中、まともに話を聞いてもらえたのは約30件
---

X投稿も作る？
```

## 例2: キャリアの記事（情報が少ない場合）

### ターン1
```
You: 転職の話書きたい
```

```
Claude: 転職か！どの転職の話？

- 空間デザイン会社 → 不動産Tech？
- 不動産Tech → リクルート？
- それとも転職全般のノウハウ？

あと、story-bankを見たけど、転職系のエピソードは
まだ【要追記】が多いね。詳しく聞かせてもらえると助かる。
```

### ターン2
```
You: 最初の転職。空間デザイン会社辞めてIT行った時の話。
7つの習慣読んで考え方変わったってのが大きかった。
```

```
Claude: あー、原点の転職ね。めっちゃ大事な話じゃん。

story-bankの [STORY-006] [STORY-007] あたりがこれだと思うけど、
まだ詳細が入ってないから、聞かせて。

1. 「7つの習慣」を読んだきっかけは？誰かに勧められた？
2. 読んで具体的に何が変わった？考え方？行動？
3. 空間デザイン会社で「もうダメだ」って思った具体的な瞬間ある？
```

### ターン3
```
You: きっかけは...確か本屋でたまたま。夜勤明けでフラフラしてて、
なんか変わりたいと思ってた時に目に入った。
読んで変わったのは「主体的である」って考え方。
環境のせいにしてた自分に気づいた。
「もうダメだ」って思ったのは...3日連続夜勤で、
朝日見ながら「俺何やってんだろ」って思った時かな。
```

```
Claude: めちゃくちゃいいエピソードだね。こんな構成でどう？

---
**タイトル案:**
3日連続夜勤の朝日。「7つの習慣」が26歳の僕を変えた話

**構成:**
1. **リード**: 今の環境に絶望してる？それ、環境のせいじゃないかもよ
2. **導入**: 朝日を見ながら「俺何やってんだろ」と思った朝
3. **本文①**: 空間デザイン会社という「憧れ」と「現実」のギャップ
4. **本文②**: 夜勤明けの本屋で出会った一冊
5. **本文③**: 「主体的である」という言葉が刺さった理由
6. **まとめ**: 環境を変える前に、自分の考え方を変えろ

---
この方向でいい？
```

---

## ポイントまとめ

### 対話のコツ
1. **最初は広く聞く**（何について書きたい？）
2. **story-bankを参照**して関連エピソードを提示
3. **具体的な数字・事実**を引き出す（何件？いつ？どのくらい？）
4. **感情・転機**を引き出す（その時どう思った？何がきっかけで？）
5. **構成提案で確認**を取ってから書く

### ユーザー側のコツ
1. **数字を意識して答える**（〜件、〜回、〜円）
2. **感情も伝える**（キツかった、嬉しかった、驚いた）
3. **「もう書いて」**と言えば質問を打ち切れる
4. **構成提案の段階で修正**を入れる（後から直すより楽）
